
# Aprendizado com o GoogleColab
 **Realizei um pequeno projeto em python, utilizando o GoogleColab uma plataforma do Google, que disponibiliza gratuitamente sua plataforma, e aproveitei para utilizar para o aprendizado.**

## Ambiente e funcionalidades 
 **As máquinas virtuais constam com um hardware de 12gb de ram e mais de 100gb de armazenamento(podendo obter mais com a versão paga).**
 **O ambiente também é vinculado ao GoogleDrive, o que facilita ainda mais a realização de projetos, facilitando a edição ou compartimento do projeto.**

### Por que utilizar o GoogleColab?
  **Escolhi utilizar a plataforma de nuvem pela sua alta capacidade de hardware, tornando o aprendizado mais fluido.**
 **Uma das escolhas também foi o fato de fazer o projeto inteiro pelo celular, e confesso que foi bem cansativo, e exigiu uma paciência enorme(mas está pronto🤓).**
